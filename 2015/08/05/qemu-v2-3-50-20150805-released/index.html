GNU ARM Eclipse QEMU 2.3.50-20150805*-dev released

Posted on August 5, 2015 by Liviu Ionescu.

Version 2.3.50-201508051820 is a maintenance partial release, adding support for Windows 64-bits.

Addressed issues

  • the _WIN64 use of sigsetjmp() was fixed and now the Windows 64-bits distribution is fully functional.

Download

The new Windows 64-bits QEMU can be downloaded from the GitHub Releases page. The packages for other platforms are available from previous releases.
